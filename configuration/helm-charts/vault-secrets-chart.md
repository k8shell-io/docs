# Vault Secrets Chart

The `vault-secrets` chart is a lightweight utility chart for defining Kubernetes Secrets whose values are pulled from [HashiCorp Vault](https://www.vaultproject.io/). It is used internally by the [k8shell-bundle chart](../k8shell-bundle-chart) and is also available standalone for injecting Vault-managed secrets into any Kubernetes workload.
