# k8shell.io

```{abstract}
K8shell is your streamlined entry point to Kubernetes, providing dynamically provisioned workspaces, both on-premise and in the cloud.
```

K8shell allows your users&ndash;developers, operations, data scientist or security architects to connect using standard SSH-based tools, ensuring a secure and isolated environment for development, testing, operations, and many other use cases. Read more about k8shell benefits and deep dive into k8shell architecture, configuration and usage in the following sections.
